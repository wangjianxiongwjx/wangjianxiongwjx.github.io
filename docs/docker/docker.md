# 虚拟化技术 -- docker

* docker help
```
//docker 版本
docker version
//docker info 信息
docker info 
docker [OPTIONS] COMMAND
```
* docker 常用命令
```java
docker images

docker images -a 
//全部镜像的ID
docker images -aq

//查询某个镜像的名称
docker search  //docker search tomcat
docker search -s 30 tomcat // star超过30 的tomcat


```
